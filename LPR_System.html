System Overview

This License Plate Recognition (LPR) system is a smart solution developed using the Raspberry Pi 5. It is designed to automatically detect, read, and record vehicle license plate numbers in real time. The system captures images through the Camera Module 2, processes the data using OpenCV for image handling, and applies Tesseract OCR to extract characters from the license plate. Recognized plate numbers are displayed on a P10 LED panel and uploaded to a cloud-based database for storage and remote access.

Hardware Used

  • Raspberry Pi 5 8 GB RAM
  • Raspberry Pi 8MP Camera Module V2
  • Adafruit RGB Matrix HAT
  • P10 LED Display Panel

Software & Libraries

  • Python
  • OpenCV
  • Tesseract OCR
  • MySQL (Cloud Database)
  • HTML, CSS, JavaScript (for Web Interface)
